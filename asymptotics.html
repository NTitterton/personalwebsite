Solving Asymptotics Questions

2018 August XX

A more formulaic approach to asymptotics questions is usually more effective than an intuitive approach. I try to bin how I approach asymptotics questions into one of six categories. Here they are in order of ascending effectiveness.

  1. Inspection
  2. Test Inputs
  3. Change in Input Size
  4. Recursion Trees
  5. Cost Model
  6. Draw a Picture

Inspection

Inspection is glancing over the code and deciding the runtime. This is what everyone tries when the first learn about asymptotics and can work for simple questions, but usually misleads or doesn't help if there is a small trick in the question.
